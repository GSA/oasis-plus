We have resources to help you navigate the OASIS+ contract; as part of this, we have included descriptions of all software applications, tools, and documentation related to the contract. If you do not see what you need, please reach out to us at [oasisplus@gsa.gov](mailto:oasisplus@gsa.gov).

## Proposal Submission and Task Management
| Resource | Description |
| --------------------------| --------------------------|
| Symphony Procurement Suite (coming soon) | The OASIS+ Program will use the Symphony procurement suite tool, also referred to as the OSP, for proposal submission, evaluations, and task order management. | 
| Training - Symphony system registration (coming soon) | Learn how to register for Symphony in a training designed for new entrants. |

| Resource | Description |

### Buyers
| Resource | Description |
| --------------------------| --------------------------|
| Buyers training - task order management (coming soon) | Learn about creating and managing your task orders in Symphony. |
| [Steps to Performance Based Acquisition](https://buy.gsa.gov/spba) | Website presenting the PBA process in easy steps, complete with samples & examples, templates, and other resources to make PBA more collaborative, performance-oriented, and team-focused.|
| [Civilian Services Acquisition Workshop](https://www.gsa.gov/buy-through-us/products-and-services/professional-services/acquisition-support/civilian-services-acquisition-workshop-csaw?gsaredirect=csaw)| Facilitated requirements development workshop built around a specific acquisition to reduce the need for contract revisions, modifications, and added costs. |

## Pricing
| Resource | Description |
| --------------------------| --------------------------|
| [Contract Awarded Labor Categories (CALC) tool](https://buy.gsa.gov/pricing/) | CALC tool helps federal contracting officers and others find awarded prices to use in negotiations for labor contracts. It offers ceiling prices, fully burdened costs, services data, and worldwide rates. |
